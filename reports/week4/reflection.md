**Week 4 Reflection**

**Learning points**
* Learned how to migrate an existing code to a new framework.
* Developed a better understanding of backend-frontend communication and how to resolve connection issues.
* Learned how to make AI actually work with user input and give useful recommendations.
* Strengthened teamwork by collaborating closely to overcome integration challenges.
* Understood that getting customer feedback early helps us build the right features.

**Validated assumptions**
* Confirmed that the AI can process photos and suggest meals based on budget.
* Validated that migrating to React improved frontend maintainability.
* Customer feedback confirmed that the new features are useful and work as expected.

**Friction and gaps**
* Backend-frontend communication turned out to be more complex than expected.
* AI agent setup took longer than we planned.
* Team coordination and communication could be improved for future sprints.

**Planned response**
* Sync up more often between frontend and backend to catch issues early.
* Start documentation earlier so it's not left until the end.
